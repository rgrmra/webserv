WebServ

Um servidor HTTP desenvolvido do zero, seguindo os princípios do RFC e permitindo a execução de CGI!

Recursos

  • Suporte a múltiplas conexões
  • Interpretação de arquivos de configuração
  • Suporte a métodos HTTP (GET, POST, DELETE)
  • Execução de scripts CGI
  • Gerenciamento de erros e respostas HTTP
